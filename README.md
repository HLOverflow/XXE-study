# XXE-study

XXE labs set up for different languages and their different parsers.
This can serve as a playground to test your scanners or use for teaching XXE.

# Available environments
- RESTFUL api
	- Python
		- eTree parser
		- lxml parser ( TODO )
	- PHP
		- DOM parser
	- Java
		- JAXB parser
		
- SOAP Web services
	- Python
	- PHP
	- Java
		- Apache Axis SOAP XXE (TODO)
